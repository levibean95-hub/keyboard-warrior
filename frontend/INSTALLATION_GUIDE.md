# Frontend Installation & Optimization Guide\n\n## 📋 Prerequisites\n\n- Node.js 18+ \n- npm 8+ or yarn 1.22+\n\n## 🚀 Quick Start\n\n### 1. Install Dependencies\n\n```bash\n# Navigate to frontend directory\ncd frontend\n\n# Install base dependencies\nnpm install\n\n# Install additional optimization packages (optional)\nnpm install --save-dev @babel/plugin-transform-react-constant-elements @babel/plugin-transform-react-inline-elements bundlesize rimraf rollup-plugin-visualizer vite-plugin-compression vite-plugin-pwa\n```\n\n### 2. Environment Configuration\n\n```bash\n# Copy environment template\ncp .env.example .env.local\n\n# Edit .env.local with your settings\n# VITE_API_URL=http://localhost:5000\n```\n\n### 3. Development Server\n\n```bash\n# Start development server\nnpm run dev\n\n# Server will start on http://localhost:3000\n```\n\n## 🏗️ Production Build\n\n### Basic Production Build\n\n```bash\n# Type check and build\nnpm run build\n\n# Preview production build\nnpm run preview\n```\n\n### Advanced Production Build (After installing optimization packages)\n\n```bash\n# Install advanced dependencies first\nnpm install --save-dev @babel/plugin-transform-react-constant-elements @babel/plugin-transform-react-inline-elements bundlesize rimraf rollup-plugin-visualizer vite-plugin-compression vite-plugin-pwa\n\n# Replace vite config with advanced version\nmv vite.config.ts vite.config.basic.ts\nmv vite.config.advanced.ts vite.config.ts\n\n# Run optimized production build\nnpm run build:production\n\n# Analyze bundle size\nnpm run build:analyze\n```\n\n## 📊 Performance Optimizations Included\n\n### ✅ Currently Active\n- **Code splitting** with manual vendor chunks\n- **TypeScript optimization** with strict checking\n- **Asset optimization** with hashing and compression\n- **React optimizations** with lazy loading\n- **Bundle size monitoring** with size limits\n- **Environment variable management**\n- **Path aliasing** for cleaner imports\n\n### 🔄 Optional (Requires Additional Dependencies)\n- **PWA support** with service workers\n- **Advanced compression** (Gzip + Brotli)\n- **Bundle visualization** with detailed analysis\n- **Babel optimizations** for React\n- **Performance monitoring** hooks\n\n## 🛠️ Available Scripts\n\n```bash\n# Development\nnpm run dev              # Start dev server\nnpm run type-check       # TypeScript validation\nnpm run lint            # ESLint checking\nnpm run lint:fix        # ESLint auto-fix\n\n# Production\nnpm run build           # Basic production build\nnpm run build:production # Advanced build with analysis\nnpm run build:analyze   # Build with bundle analyzer\nnpm run preview         # Preview production build\n\n# Maintenance\nnpm run clean           # Clean dist folder\nnpm run bundle-size     # Check bundle sizes\n```\n\n## 📁 Project Structure\n\n```\nfrontend/\n├── public/              # Static assets\n├── src/\n│   ├── components/\n│   │   ├── common/      # Reusable components\n│   │   ├── core/        #
